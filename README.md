## What I do

I build software for hard problems: planning, scheduling, optimization, AI orchestration, and the systems around them.

My work sits at the intersection of:
- **Constraint solving & optimization**
- **AI agents and applied machine learning**
- **Developer tools and operator systems**
- **Production-grade backend, infrastructure, and cloud systems**

I care about software that is not just technically interesting, but operationally real: fast, reliable, and useful in production.
